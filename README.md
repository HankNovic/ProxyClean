# ProxyClean

本仓库用于公开可用 SOCKS5 代理数据及相关说明。

## 公开内容

- `good_socks.txt`：清洗后的可用 SOCKS5 代理列表。

## 代理池特点

- **多源聚合清洗**：对公开来源进行聚合、去重与有效性筛选。
- **可用优先**：仅保留通过连通性验证的代理节点。
- **持续更新**：代理列表会定期滚动更新，失效节点会被淘汰。
- **时效性导向**：尽量保证列表反映近期可用状态，但不保证长期稳定。

## 代理池文件格式

`good_socks.txt` 格式如下：

```text
ip:port
```

- 每行一个 SOCKS5 代理。
- 使用英文冒号 `:` 分隔 IP 与端口。
- 默认不包含用户名/密码信息。

## 使用方式

下载 `good_socks.txt` 后，按行读取并选择代理使用，例如：

```text
1.2.3.4:1080
```

在命令行测试（`curl` 示例）：

```bash
curl --socks5 1.2.3.4:1080 https://api.ipify.org
```

如需让域名解析也走代理：

```bash
curl --socks5-hostname 1.2.3.4:1080 https://api.ipify.org
```
