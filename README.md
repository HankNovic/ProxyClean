# ProxyClean

本仓库用于公开可用代理数据及相关说明。

## 公开内容

- `good_socks.txt`：清洗后的可用 SOCKS 代理列表（每行一个 `ip:port`）。

## 数据说明

- 数据基于持续清洗与有效性校验后定期更新。
- 对外提供数据来源说明、清洗说明与测试地址信息（如已发布）。
- 仅用于学习与研究交流，请遵守所在地法律法规及目标服务条款。

## 使用方式

直接下载 `good_socks.txt`，按行读取使用即可。

## 说明

- 代理可用性具有时效性，实际连通性请以使用时测试结果为准。
- 若文档中提供测试地址或验证方式，请以文档最新说明为准。
